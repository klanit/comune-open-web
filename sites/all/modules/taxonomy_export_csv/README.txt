DESCRIPTION:
------------
This module provides an taxonomy terms export in csv format for migration.


INSTALLATION:
-------------
1. Goto your Drupal site, path: sites/all/modules.

2. Enable the "Taxonomy export csv" module by navigating to:

     administer > modules
